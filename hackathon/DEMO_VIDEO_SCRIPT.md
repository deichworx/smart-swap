## Recording Tips

### Screen Recording
- Use Android: `scrcpy` with `--bit-rate 8M`
- Use iOS: QuickTime Player screen recording
- Record at 60 FPS if possible

### Device Setup
- Full battery or charging
- Do Not Disturb mode ON
- Hide notification bar if possible
- Clean up recent apps

### Demo Data
- Pre-load wallet with test tokens
- Have some swap history ready
- Set SKR balance to show mid-tier progress

### Voiceover
- Record separately in quiet room
- Use good microphone (even AirPods work)
- Speak clearly, moderate pace
- Record multiple takes

### Editing Tools
- DaVinci Resolve (free)
- CapCut (mobile)
- iMovie (Mac)
- Adobe Premiere

### Music
- Epidemic Sound (paid, royalty-free)
- YouTube Audio Library (free)
- Keep it subtle, don't overpower voiceover
